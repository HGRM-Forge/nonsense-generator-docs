## Main dependencies

-   **Spring Boot**: A framework for rapid development of Java applications based on Spring, providing auto-configuration and integrated tools for building web services and REST APIs.
-   **Lombok**: A library that reduces boilerplate in Java code by automatically generating methods like getters, setters, constructors, and more through annotations.
-   **Google Cloud Language SDK**: An SDK for integrating with Google Cloud's Natural Language Processing services, useful for text analysis and toxicity moderation.
-   **Jackson**: A library for serializing and deserializing Java objects to and from JSON, widely used for handling JSON data.
